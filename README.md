# Smart Tourism Management System

## English | [中文](#中文)

A smart tourism and scenic area management system built with Go, Gin, PostgreSQL, and Redis.

### Features
- Scenic spot management
- Ticket booking and sales
- Visitor flow monitoring and control
- Tour guide scheduling
- Hotel and accommodation booking
- Route planning and recommendations
- Event and activity management
- Visitor feedback and complaints
- Revenue analytics
- Emergency management

### Quick Start
```bash
docker-compose up -d
go run cmd/api/main.go
```
